Delete Files

The delete file command executes when you click on the edit menu and then select the file option from the delete submenu. A window frame then displays to allow you to traverse folders and locate the file you want to delete. Double click on the file name that displays or type its name and click the OK button. This action will cause the file to be removed from the system. You can click the cancel button to abort this command.

Additional Notes

Be careful how you use this command. Deleting system files can damage your system.
