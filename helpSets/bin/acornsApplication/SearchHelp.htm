Help Search

When you execute the Help command, three tabs show on the left side of the display. One of these tabs has the label 'Search'. If you select this option, you can search for a term in the help database by entering your search criteria. The help system then displays a list of all matching pages.

If you click on one of the bulleted items, the help system displays detailed information in the right panel of the Help Screen. The search option operates similar to the way that web-based search engines work.
