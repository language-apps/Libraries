File Properties

The file properties command executes when you click the properties option on the file menu. When you execute this command, a dialogue appears requesting file header information. None of this information is required, but it can be useful. We describe the parts of this header information below.

  1. Language: The language or languages to which the lessons in this file apply.
  2. Author: The name of the author who is creating this file.
  3. Description: Any descriptive information pertaining to this file.
