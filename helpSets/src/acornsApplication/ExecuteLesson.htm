Play (Execute) a Lesson

To 'play a lesson' click on the play icon on the system toolbar or select the execution option from the toolbar menu. When this command executes, a full screen view of the lesson displays and a lesson executes according to its design. Note that different types of lessons execute differently. Please refer to the Lesson help pages for more information.

All lessons have a standard row of buttons that display at the bottom when the lesson executes. Please follow this link for more information.

Additional Notes

Some lessons cannot execute if they are not properly setup. For example Picture and Sound lessons cannot execute until a picture (image) has been added.

Other lessons require audio. Refer to Maintaining and Recording Audio for more information.
