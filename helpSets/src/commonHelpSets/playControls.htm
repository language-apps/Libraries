Play Mode Control Buttons

At the bottom of an executing lesson is a series of standard control buttons as shown in the picture below. The table that follows the picture describes the purpose of each of these buttons.

Next Layer
 Pressing this button will move to the next higher layer. ACORNS layers can be setup for many reasons. For example, they can represent different dialects or difficulty levels. If a lesson only has one layer, this button will have no effect.
Previous Layer
 Pressing this button will move to the next lower layer. ACORNS layers can be setup for many reasons. For example, they can represent different dialects or difficulty levels. If a lesson only has one layer, this button will have no effect.
Scroll to Linked Lesson
 Lessons can be setup to link between lessons in a file. This button appears when the option is active. Clicking on this button will switch you to the linked lesson. If the link is not valid, the button will show, but the option won't work.
Trigger Popup Options
 Pressing this button will cause a popup menu to appear. Different lesson types have different options that users can choose from.
Scroll to First Lesson
 Clicking this button enables you to scroll to the first lesson in the ACORNS file. If you are already at the first lesson or if your file only contains one lesson, nothing will happen.
Scroll to Previous Lesson
 Clicking this button enables you to scroll to the previous lesson in the ACORNS file. If you are already at the first lesson, a beep will sound.
Scroll to Next Lesson
 Clicking this button enables you to scroll to the next lesson in the ACORNS file. If you are already at the last lesson, a beep will sound.
Scroll to Last Lesson
 Clicking this button enables you to scroll to the last lesson in the ACORNS file. If you are already at the last lesson or if your file only has one lesson, nothing will happen.
Record
 If you click this button and have a microphone available, you can record what you heard when a lesson outputs audio. In this way you can check what you heard against how you might say it.
Play Back
 After you record some audio, this button lets you play it back to see how it sounds.
Stop Recording
 When you stop recording, you should click this button to tell ACORNS that you are finished.
Help
 Click this button to get help regarding executing the current lesson.
